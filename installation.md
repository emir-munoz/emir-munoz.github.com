# Installation

# Pre-requirements

- Ruby
```bash
$ sudo apt install ruby
$ sudo apt install ruby-dev
```

- Jekyll
```bash
$ sudo gem install jekyll
```

- Jekyll-scholar
```bash
$ sudo gem install jekyll-scholar
```

- Pygments
```bash
$ sudo pip install Pygments
```

(Currently, changed to `rouge` which has out-of-the-box support.)
